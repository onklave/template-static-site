About

This is a static-site starter template. There is no server and no runtime — the build produces a folder of static assets. Edit the markup, styles, and TypeScript to make it yours.

Get started

Clone the template, install dependencies, and start the dev server. When you are ready to ship, run the build and Onklave serves dist/ from the edge. 

npm ci
npm run dev      # develop locally
npm run build    # build to dist/

Built with the Onklave static-site template.
